s.3)Bir mağazada alınan ürünün fiyatına göre kargo ücreti eklenmektedir.
Girilen fiyat bilgisine göre ödenecek toplam tutarı yazınız.
Kargo ücreti 100 liraya kadar 10 TL, 100-200 arası 5 liraya
200 üzerikargo bedeli yok.

int toplamFiyat = 0;
Console.WriteLine("Fiyat giriniz:");
int urunFiyati = Convert.ToInt32(Console.ReadLine());
if (urunFiyati < 100)
{
	toplamFiyat = urunFiyati + 10;
}
else if (urunFiyati >= 100 && urunFiyati <= 200)
{
	toplamFiyat = urunFiyati + 5;
}
else
{
	toplamFiyat = urunFiyati;
}
Console.WriteLine("Ödenecek toplam tutar:" + toplamFiyat);
Console.Read();

s.4) Girilen 15 sayıdan kaç tanesinin pozitif, kaç tanesinin negatif
sayı olduğunu yazan programı yazınız.

	int pozitifSayiMiktari=0,negatifSayiMiktari=0;
	for (int i=1;i<16;i++){
		Console.WriteLine(i+". Sayıyı giriniz:");
		int sayi=Convert.ToInt32(Console.ReadLine());
		if(sayi>0)
		{pozitifSayiMiktari++;}
